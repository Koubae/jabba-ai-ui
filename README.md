### Quick Start

*  Install

```bash
make init
```

* Run

```bash
make run
```

Go to http://localhost:3000


### Deployment

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.


### Docker

```bash
# build
make build
# run all 3 services
make run 
```

```bash
# Locally preview production build
npm run preview
npm run build
```
